## 🧩 Projetos em Destaque

### 📌 Sistema de Cobrança Digital

**Descrição:** sistema para gestão de cobrança digital com envio de e-mails, mensagens via WhatsApp, controle de contatos, histórico de interações e apoio ao trabalho dos operadores.

**Problema:** empresas muitas vezes não abrem e-mails, não respondem, não atendem telefone ou não retornam contatos. Isso dificulta o acompanhamento manual e reduz a produtividade da equipe.

**Solução:** estrutura de automação e organização de contatos para disparar notificações, registrar interações, centralizar dados e facilitar a tomada de decisão dos operadores.

**Tecnologias:** PHP, Node.js, MySQL, AWS SES, WhatsApp API, Docker, APIs REST.

---

### 📌 Integração com AWS SES e Webhooks

**Descrição:** backend para envio e monitoramento de e-mails usando Amazon SES, com tratamento de eventos como delivery, bounce, complaint, open e click.

**Problema:** disparos em volume precisam de controle de reputação, rastreamento de eventos, segurança de credenciais e análise de falhas de entrega.

**Solução:** criação de rotas, webhooks e processamento de eventos para acompanhar a saúde dos envios e melhorar a confiabilidade da comunicação.

**Tecnologias:** Node.js, Express, AWS SES, SNS, Webhooks, Docker.

---

### 📌 Automações com n8n e Inteligência Artificial

**Descrição:** fluxos automatizados para leitura de dados, integração com APIs, processamento de informações e respostas assistidas por IA.

**Problema:** tarefas repetitivas consomem tempo dos operadores e aumentam o risco de erro manual.

**Solução:** criação de automações conectando CSV, APIs, banco de dados, sistemas internos e modelos de IA para acelerar tarefas operacionais.

**Tecnologias:** n8n, IA, APIs REST, Webhooks, CSV, Banco de Dados.

---

### 📌 Modernização de Sistemas Legados

**Descrição:** modernização gradual de sistemas antigos em PHP para arquiteturas mais organizadas usando React.js, Next.js, Node.js e Tailwind CSS.

**Problema:** sistemas legados dificultam manutenção, evolução, segurança e experiência do usuário.

**Solução:** refatoração por etapas, separação entre frontend e backend, padronização de rotas, criação de componentes reutilizáveis e melhoria da interface.

**Tecnologias:** PHP, React.js, Next.js, Node.js, Tailwind CSS, MySQL, Axios.
